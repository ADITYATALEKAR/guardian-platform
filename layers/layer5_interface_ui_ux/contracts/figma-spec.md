# AVYAKTA LAYER 5 — FIGMA DESIGN CONTRACT

This document defines the visual governance rules
for designers working on Layer 5.

This is NOT a creative playground.
This is an operational system.

---

## Typography

Font: IBM Plex Sans
Data/ID Font: IBM Plex Mono

H1: 20px / 600  
H2: 16px / 600  
H3: 14px / 600  
Body: 14px / 400  
Secondary: 13px / 400  
Caption: 12px / 400  
Meta-micro: 11px / 400 (timestamps, GIDs, cycle IDs)

No text above 20px.

---

## Control Heights

Small: 24px  
Default: 30px  
Large: 40px  

All buttons and inputs align to 30px.

---

## Button Rules

Padding left/right: 10px  
Border radius: 3px  
Icon size: 16px  
Icon gap: 6px  

Only one primary button per group.

---

## Colors

App background: #000000  
Panel background: #0a0a0a  
Border: #1c1f24  

Primary: #2b95d6  
Success: #15b371  
Warning: #f29d49  
Danger: #f55656  
Unknown/Not observed: #4A5568

Quantum badge states:
- Ready: #00C853
- Not ready: #FF6D00
- Unknown: #546E7A

No gradients.
No soft glow.
No neon tones.

---

## Logo Governance

Logo appears only in primary header.
Height: 24px.
Never repeated.

---

## Design Philosophy

- Data first.
- No decorative elements.
- No visual noise.
- No startup aesthetic.
- Enterprise seriousness.
